# Armory

Armory is a React application built with TypeScript and Vite. It provides a modern development experience with fast hot module replacement (HMR), strict type checking, and a clean ESLint setup tailored for React projects.

## Project Overview

Armory is designed to serve as a foundation for building a frontend application with React and TypeScript. It includes the standard Vite development workflow and recommended tools for maintaining code quality and performance.

Key goals:
- fast local development with Vite and HMR
- type-safe React components using TypeScript
- consistent project structure for scalability
- linting and formatting guidance for production-ready code

## Technologies Used

- React 18+
- TypeScript
- Vite
- ESLint
- Optional React Compiler support
- Recommended Vite React plugin options



## Project Structure

- `src/` - application sources
- `src/main.tsx` - application entry point
- `src/App.tsx` - root app component
- `src/assets/` - static assets
- `public/` - public files served as-is
- `tsconfig.app.json` - TypeScript configuration for the app
- `tsconfig.node.json` - TypeScript configuration for node tooling
